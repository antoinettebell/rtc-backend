# Food truck Backend

It is a backend project having node along with mongoDB


## Installation

Copy `.env.example` and rename it to `.env`

Use package manager to install

```
  npm install
``` 

Server will be started on port 3000

Socket Server will be started on port 3001

Just go to your browser and browse to

```
http://localhost:3000
```
And You will see below message
```json
{
  "code": 200,
  "success": true,
  "data": null,
  "error": null,
  "message": "'food-truck' started"
}
```
